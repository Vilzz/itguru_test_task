## 🚀 React + TypeScript + Vite — Инструкция по запуску

#### 📦 1. Клонирование репозитория

```
git clone https://github.com/Vilzz/itguru_test_task.git
```

```
cd itguru_test_task
```

#### 🧩 2. Убедитесь, что установлен Node.js (рекомендуемая версия: 18 или выше):

```
node -v
```

#### 📥 3. Установка зависимостей

```
npm install
```

#### ▶️ 4. Запуск проекта

```
npm run dev
```
